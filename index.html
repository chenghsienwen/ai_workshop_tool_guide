開啟 PowerShell（以系統管理員身分）

很多安裝步驟需要用到 PowerShell。開啟方式：

開啟 PowerShell 步驟

1. 按下鍵盤上的 Windows 鍵

2. 輸入 PowerShell

3. 在「Windows PowerShell」上按右鍵 → 以系統管理員身分執行

⚠️ 請勿使用「命令提示字元 (CMD)」或「Git Bash」，否則部分指令可能無法執行。

步驟 1：安裝 Node.js（LTS 版本）
Codex CLI 與 Claude Code 的執行環境 · 約需 5 分鐘
🏢
公司電腦用戶請注意：需先向 IT 申請安裝權限
公司管理的 Windows 電腦通常限制自行安裝軟體。請在 Workshop 前 2～3 個工作天向 IT Help Desk 提交申請，避免當天無法安裝。

📋 IT Ticket 建議填寫內容：

主旨：申請安裝 Node.js LTS — AI Workshop 使用

內容：

您好， 我將參加 [日期] 的 AI Workshop， 課程需要在本機安裝以下軟體： - 軟體名稱：Node.js LTS - 下載來源：https://nodejs.org（官方網站） - 用途：執行 AI CLI 工具（Codex CLI / Claude Code） - 需要權限：本機系統管理員安裝權限 請問能否協助開放安裝權限，或由 IT 協助預先安裝？ 謝謝！
1

前往 Node.js 官方網站

用瀏覽器打開以下網址：

https://nodejs.org

你會看到兩個下載按鈕：

✅ LTS（推薦）
22.x.x
Current
不建議

👆 請點選左邊綠色的「LTS」按鈕下載

💡 為什麼選 LTS？ LTS（長期支援版）更穩定，與大部分 AI 工具的相容性最好。
2

執行安裝程式

下載完成後，雙擊 .msi 安裝檔，然後：

  • 點「Next」→「Next」→ 接受授權條款
  • 安裝路徑保持預設，直接 Next
  • 出現「Tools for Native Modules」頁面，不用勾選，直接 Next
  • 點「Install」→「Finish」
🖼️ Node.js 安裝精靈畫面（選擇 LTS、一路按 Next 即可）
3

驗證安裝成功

安裝完成後，關閉並重新開啟 PowerShell，輸入：

PowerShell
PS> node -v
# 應顯示類似 v22.x.x（參考版本：v22.14.0）

PS> npm -v
# 應顯示類似 10.x.x
✅ 看到版本號碼代表 Node.js 已成功安裝！

步驟 3：安裝 Python
AI 腳本執行環境 · 參考版本：Python 3.10.12 · 約需 5 分鐘
🏢
公司電腦用戶請注意：需先向 IT 申請安裝權限
請在 Workshop 前 2～3 個工作天向 IT Help Desk 提交申請，並在 IT Ticket 中加入以下資訊：

📋 IT Ticket 建議填寫內容（Python）：

- 軟體名稱：Python 3.x（官方版） - 下載來源：https://www.python.org（官方網站） - 用途：執行 AI Workshop 課程的 Python 腳本 - 需要權限：本機系統管理員安裝權限
1

前往 Python 官方網站下載

用瀏覽器打開以下網址：

https://www.python.org/downloads/

頁面頂部會有黃色的大型下載按鈕：

⬇ Download Python 3.x.x

它會自動偵測你的 Windows 版本並提供對應的安裝檔

💡 建議下載 3.10.x 或更新的穩定版，本指南的參考版本為 Python 3.10.12。
2

執行安裝程式（重要：勾選 Add to PATH）

雙擊下載的 .exe 安裝檔，在安裝畫面的最下方務必勾選：

📋 Python 安裝器首頁（重要！）

勾選後點「Install Now」，等待安裝完成再點「Close」。

⚠️ 若忘記勾選「Add Python to PATH」，需要重新安裝，或手動將 Python 路徑加入環境變數。
3

驗證安裝成功

安裝完成後，關閉並重新開啟 PowerShell，輸入：

PowerShell
PS> python --version
# 應顯示類似 Python 3.x.x（參考版本：Python 3.10.12）

PS> pip --version
# 應顯示 pip 版本號
💡 若 python 無法執行，請嘗試 python3 --version。Windows 有時會將指令名稱設為 python3。
✅ 看到版本號碼代表 Python 已成功安裝！

步驟 4：安裝 Codex CLI（OpenAI）
免費 · 需要 ChatGPT 帳號 · 約需 3 分鐘
✅ 完全免費！ 用 ChatGPT 免費帳號即可登入使用 Codex CLI，無需信用卡。
🏢
公司電腦用戶請注意：網路防火牆可能封鎖 Codex CLI
即使安裝成功，公司的網路安全設定可能會在你嘗試登入時顯示以下錯誤畫面。若出現這些症狀，請聯繫 IT Help Desk 申請開放網路存取權限。

症狀 1：登入時瀏覽器顯示「Sign-in could not be completed」

原因：公司防火牆封鎖了 OpenAI OAuth 認證伺服器（auth.openai.com）的連線。

症狀 2：執行時彈出「Restricted Access」對話框

原因：公司端點安全軟體（如 Symantec / CrowdStrike）封鎖了 codex.exe 的對外連線。

📋 IT Ticket 建議填寫內容（Codex CLI 網路權限）：

主旨：申請開放 Codex CLI 網路存取 — AI Workshop 使用 您好， 我在使用 AI Workshop 的 Codex CLI 工具時， 遭遇公司安全政策封鎖，出現以下錯誤： - 錯誤 1：瀏覽器顯示「Sign-in could not be completed」 （token_exchange_failed，無法連線至 auth.openai.com） - 錯誤 2：端點安全軟體顯示「Restricted Access」 （封鎖 codex.exe 對外連線至 185.199.111.133:443） 需要開放的網路存取： - auth.openai.com（OAuth 認證） - api.openai.com（API 呼叫） - 應用程式：codex.exe（OpenAI OpCo, LLC） 謝謝！
1

以系統管理員身分開啟 PowerShell

請確保使用 PowerShell（不是 CMD 或 Git Bash）。

2

執行安裝指令

在 PowerShell 中貼上並執行以下指令：

PowerShell（以系統管理員執行）
powershell -ExecutionPolicy ByPass -c "irm https://chatgpt.com/codex/install.ps1 | iex"
💡 安裝完成後，關閉並重新開啟 PowerShell，讓 PATH 設定生效。
3

驗證版本並首次登入

# 應顯示類似 codex-cli x.x.x（參考版本：codex-cli 0.142.2）

確認版本後，輸入 codex 啟動並登入：

PowerShell
PS> codex

首次執行時，會提示你登入 ChatGPT 帳號，瀏覽器會自動開啟讓你完成授權。登入後即可開始使用！

Codex CLI 使用方式

常見問題排除
❓ 輸入指令後出現「不是內部或外部命令」

PATH 設定尚未生效。請完全關閉 PowerShell 後重新開啟，不是按 X 最小化，而是關閉後重開。如果還是不行，請重新啟動電腦。

❓ npm 安裝出現「permission denied」錯誤

請確保以系統管理員身分開啟 PowerShell（右鍵點選 → 以系統管理員身分執行）。

❓ Claude Code 出現「Raw mode is not supported」

請確保你使用的是 PowerShell，而非 Git Bash 或 CMD。Claude Code 的互動介面需要 PowerShell 的 TTY 支援。

❓ Antigravity CLI 登入後瀏覽器無法開啟

手動複製終端機顯示的 URL，貼到瀏覽器網址列即可完成授權。

❓ Antigravity CLI 登入時 Google OAuth 授權失敗，或出現「Restricted Access」

這是公司網路安全設定造成的封鎖，不是你的操作問題。需要 IT 協助開放以下存取權限：

  • accounts.google.com — Google OAuth 登入
  • oauth2.googleapis.com — OAuth Token 交換
  • generativelanguage.googleapis.com — Gemini API 呼叫
  • agy.exe 應用程式的對外網路存取

請參考 Antigravity CLI 安裝步驟中的 IT Ticket 範本，將錯誤截圖一併附上送出。

❓ Codex CLI 登入時顯示「Sign-in could not be completed」或「Restricted Access」

這是公司網路安全設定造成的封鎖，不是你的操作問題。需要 IT 協助開放以下存取權限：

  • auth.openai.com — OAuth 登入認證
  • api.openai.com — Codex CLI API 呼叫
  • codex.exe 應用程式的對外網路存取

請參考 Codex CLI 安裝步驟中的 IT Ticket 範本，將錯誤截圖一併附上。
